## 📝 开发规范

### 后端代码结构

```
backend/src/
├── routes/          # 路由定义
├── controllers/     # 请求处理
├── services/        # 业务逻辑
├── middleware/      # 中间件
└── utils/           # 工具函数
```

### 命名规范

- 服务文件: `xxxService.ts`
- 控制器文件: `xxxController.ts`
- 路由文件: `xxxRoutes.ts`

### 提交规范

```
feat: 添加借阅功能
fix: 修复登录问题
docs: 更新文档
refactor: 重构用户服务
```
